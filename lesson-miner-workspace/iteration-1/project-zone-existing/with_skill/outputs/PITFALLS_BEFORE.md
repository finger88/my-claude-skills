## 卡点记录

### 1. 飞书 webhook Token 频繁过期

| 维度 | 内容 |
|------|------|
| **现象** | 调用飞书 API 返回 401/403 错误，消息发送失败 |
| **根因** | 飞书自定义机器人的 webhook token 有有效期限制，默认 3 个月 |
| **解决方案** | 1. 定期在飞书群设置中重新生成 webhook URL<br>2. 将 token 提取为环境变量，不硬编码<br>3. 添加 token 过期检测逻辑，提前预警 |
| **检测信号** | API 返回 401/403 + 之前正常运行的代码突然失效 |

### 2. 食材图片识别受光线影响大

| 维度 | 内容 |
|------|------|
| **现象** | 用户上传同一道菜的照片，不同光线条件下识别结果差异大 |
| **根因** | 图像识别模型对光线敏感，阴影/反光导致特征提取不稳定 |
| **解决方案** | 1. 引导用户在自然光下拍摄<br>2. 添加图像预处理（亮度均衡、去噪）<br>3. 提供"重新识别"按钮让用户手动触发 |
| **检测信号** | 识别置信度低于 0.7 + 图片有明显阴影或反光 |
